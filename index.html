ChitChat

Real-time chat app built with Node, Express, MongoDB, React, Redux, and Socket.io. Inspired by Slack

  • Users can create and view different channels, along with adding specific users to join in on the discussion. Multi-person private messaging is also available
  • Real-time, fully customizable messaging is featured
  • 3rd party API integration via GIPHY allows for smart gifs that can transform users' words into gifs
  • Add emojis to messages or have emoji reactions to other users' messages

Quicknote

Full stack note-taking application inspired by Evernote. Built with React, Redux, Ruby on Rails, and PostgreSQL.

  • Users can use a rich text editor (implemented with Quill.js) to allow for flexible and robust note taking
  • Notes can be organized via notebooks, which can be created and viewed in a clean UI
  • Tags can organize notes across different notebooks
  • Notes are autosaved for a smoother user experience
